# ToDoList
A lightweight offline to-do list app — single HTML file, no dependencies, works on any browser across desktop &amp; mobile.
A dead-simple, fully offline to-do list built with vanilla HTML, CSS, and JavaScript — no frameworks, no install, no internet required. Just open the .html file in any browser and start adding tasks. Data persists via localStorage so your tasks survive refreshes and browser restarts. Works on both desktop and mobile.
